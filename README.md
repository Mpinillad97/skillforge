# SkillForge

> AI-powered platform that helps software engineers analyze job opportunities, identify skill gaps, and build personalized learning roadmaps.

## Overview

SkillForge is a platform designed for developers who want to improve their careers through data-driven insights.

Instead of simply tracking job applications, SkillForge analyzes job descriptions, compares them against your profile, identifies missing skills, and generates personalized learning paths.

The long-term goal is to become an all-in-one career platform for software engineers.
