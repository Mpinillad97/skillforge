# ADR-001: Initial Technology Stack

## Status

Accepted

---

## Context

SkillForge requires a backend capable of handling APIs,
AI integrations and future scalability.

---

## Decision

We will use:

Backend:
- Python
- FastAPI

Database:
- PostgreSQL

Frontend:
- React + TypeScript

Infrastructure:
- Docker

---

## Reasoning

FastAPI provides:

- Strong typing
- High performance
- Excellent AI ecosystem
- Good developer experience

PostgreSQL provides:

- Reliability
- Strong relational modeling
- Industry adoption
