 # Technical Architecture

## Overview

SkillForge will start as a modular monolith architecture.

The first version will prioritize maintainability and fast iteration.

---

## Components

### Frontend

Responsible for:

- User interface
- Dashboard
- Job management
- Profile management


### Backend API

Responsible for:

- Authentication
- Business logic
- Data management
- AI orchestration


### Database

PostgreSQL stores:

- Users
- Profiles
- Jobs
- Applications
- Skills


### AI Layer

Responsible for:

- Job analysis
- Skill extraction
- Recommendations
